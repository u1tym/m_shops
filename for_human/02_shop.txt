確認したい点
仕様として次のように決めています。問題なければこのまま実装に進めます。

1.
店舗更新は PUT（全置換） — 子要素も送った配列で丸ごと差し替え。部分更新の PATCH は初版では設けていません。
2.
検索の q 指定時 — station / location 等は無視し、q だけで横断検索します。
3.
地図の複数店舗マーカー — API は各店の google_maps_url を返すのみ。1 画面に複数マーカーはフロント＋ Maps JS API（初版 API 範囲外）としています。
変更の希望があれば教えてください。問題なければ backend の FastAPI 実装に進めます。
